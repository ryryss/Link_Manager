There is a tool called Link Manager, create by claude code and chatgpt, use python.

Just drop the *.csv to the window, it will generate a file named links.xlsx in the same directory as the local executable file, automatically copy the CSV data into it, and automatically remove duplicates.

Select a line and press 'n'. The line will turn red. Pressing 'y' will turn it green, indicating whether the link has been sold.

In addition to using color to mark sold links, you can also use the delete key on your keyboard to directly delete selected rows.

It also supports copy and cut functions (ctrl + c or v). If you use the cut function, you will be prompted whether to delete.

All code runs locally, without internet connectivity, so your data is secure.
